DIT IP Info Tool Documentation

Usage

The DIT IP Info cli Tool is a Python script that allows users to retrieve IP-related information, perform network tasks such as pinging and Nmap scans, and get details about web applications. It provides a simple command-line interface for these functionalities.

Supported Operating Systems

This tool is compatible with Windows, macOS, and Linux operating systems.

Installation

You can install the tool using the following steps:

  1. Clone the repository from GitHub:
    2. git clone https://github.com/Dit-Developers/Dit-IP-Info-cli
  2. Navigate to the cloned directory:
    3. Dit-IP-Info-cli-main
  3. Run the script:
    4. python3 dit_ip_info.py

How It Works

The tool uses various Python libraries such as requests, subprocess, os, socket, nmap, and rich to perform its functions. Here's an overview of its main functionalities:

  • Retrieve IP Information
    • Get your own IP address.
    • Retrieve detailed information about any IP address using the ip-api.com service.
  • Network Tasks
    • Ping an IP address to check its availability.
    • Get the IP address of a website.
  • Local Network
    • Get your local IP address.
  • Nmap Scanning
    • Perform an Nmap scan on an IP address.
  • Web Application Enumeration
    • Perform enumeration tasks on web applications (you can replace this with your own script).

Commands

  • 1: Get your IP address.
  • 2: Retrieve detailed information about any IP address.
  • 3: Information about the tool developer.
  • 4: Ping an IP address.
  • 5: Get the IP address of a website.
  • 6: Get your local IP address.
  • 7: Perform an Nmap scan on an IP address.
